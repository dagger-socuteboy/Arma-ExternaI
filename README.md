## 👁️ Overview



### A specialized visual modification designed to render distinct, high-contrast chameleon textures on enemy character models within Arma Reforger. This tactical utility ensures players can instantly identify and track hostile combatants across complex terrain and through visual obstructions.



## ⚡ Key Features



### * 🎯 **Precision:** Accurately maps brightly colored textures exclusively to enemy player models without distorting the surrounding environment or friendly units.

### * 🛡️ **Security:** Operates entirely within client-side memory boundaries to minimize detection vectors by standard anti-cheat algorithms.

### * 🚀 **Performance:** Features highly optimized rendering hooks that maintain stable frame rates even during intense, large-scale firefights.
